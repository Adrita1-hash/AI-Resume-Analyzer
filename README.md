# AI Resume Analyzer and ATS Matching System

## Overview
This project is an AI-powered Resume Analyzer built using Python, NLP, TF-IDF, and Streamlit. It analyzes resumes, compares them with job descriptions, calculates ATS match scores, and identifies missing skills.

---

## Features
- Resume PDF Upload
- PDF Text Extraction
- NLP-based Text Cleaning
- Skill Detection
- ATS Match Score Calculation
- Missing Skill Detection
- Interactive Streamlit Web App

---

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- NLTK
- PDFPlumber
- TF-IDF
- Cosine Similarity

---

## How It Works
1. Upload Resume PDF
2. Paste Job Description
3. System extracts resume text
4. NLP processing is applied
5. Skills are detected
6. ATS score is calculated
7. Missing skills are displayed
